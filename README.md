# UNNG APT

Static APT repository for UNNG Debian and Ubuntu packages, served by GitHub Pages at `https://apt.unng.ru`.

Add the repository on Debian/Ubuntu:

```sh
sudo install -d -m 0755 /etc/apt/keyrings
curl -fsSL https://apt.unng.ru/apt/unng.gpg | sudo tee /etc/apt/keyrings/unng.gpg >/dev/null
echo "deb [signed-by=/etc/apt/keyrings/unng.gpg] https://apt.unng.ru/apt stable main" | sudo tee /etc/apt/sources.list.d/unng.list
sudo apt update
```

Install a package, for example:

```sh
sudo apt install endlessnet-client
```

Repository contents under `apt/` are generated by publishing workflows from UNNG project repositories.
