EndlessNet APT

APT repository for EndlessNet Debian and Ubuntu packages.

sudo install -d -m 0755 /etc/apt/keyrings
curl -fsSL https://apt.unng.ru/apt/endlessnet.gpg | sudo tee /etc/apt/keyrings/endlessnet.gpg >/dev/null
echo "deb [signed-by=/etc/apt/keyrings/endlessnet.gpg] https://apt.unng.ru/apt stable main" | sudo tee /etc/apt/sources.list.d/endlessnet.list
sudo apt update
sudo apt install endlessnet-client

Repository metadata is published under /apt/dists/stable/.
